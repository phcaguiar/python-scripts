## Setup

Before all, let's install the dependecies. Make sure you're using Python 2.7, then run:

```
$ pip install pipenv
$ pipenv install --three
$ export personal_access_token=''
$ export slacktoken=''
$ export base_url=''
$ export organization_name=''
$ export project_name=''
$ export build_id=''
$ export kvurl=''
$ export certname=''
```
